# OOP-Lab7-
# Airline Management System - Lab 7

## 📌 Topshiriq haqida
CS 210 Computer Programming 2 - Polymorphism in Java (JavaScript)

## ✅ Bajarilgan talablar
- ✅ Polymorphism (Customer extends Person, FlightSeat extends Seat)
- ✅ Flight search
- ✅ Reservation with multiple passengers
- ✅ Itinerary (multi-flight)
- ✅ Payment processing
- ✅ Cancellation
- ✅ Admin functions

## 📂 Fayllar
- enums.js - Enumlar (FlightStatus, ReservationStatus)
- models.js - Barcha classlar (Person, Customer, Aircraft, Seat, FlightSeat)
- flights.js - Flight va FlightInstance classlari
- reservations.js - Reservation va Itinerary classlari
- main.js - Asosiy kod (demo)

## 🚀 Ishga tushirish
bash
node main.js
